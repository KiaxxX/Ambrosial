## 使用指南

### 小米运动自动刷步数(Python/Funcs/ExStep)
1. 变量名为  **SCKEY**、**USER**、**PWD**、**STEP** 的变量 \
值分别为 **server酱推送key**、 **账号(仅支持手机号)**、**密码**、**步数(49980-89980,0表示10000-20000)**
2. 每天运行一次,在下午6点较好
